# ثيرا (Thera)

**ثيرا** منصة ويب للصحة النفسية باللغتين **الإنجليزية والعربية**: دعم إرشادي بالذكاء الاصطناعي، اكتشاف المعالجين، حجز الجلسات، الدفع (بطاقة عبر Polar أو تحويل بنكي عبر روابط إنستاباي التي يضيفها المعالج)، جلسات فيديو (Daily.co)، ولوحات تحكم حسب الدور للبالغين وأولياء الأمور والمراهقين والمعالجين والمسؤولين.

يُقدَّم المشروع لمراجعة **Edu Tech** و**We Schools** كمشروع ختامي لفريق **Semi Colon Team** (فريق السيمي كولون).

## المكدس التقني (ملخص)


| المجال             | التقنيات                                                     |
| ------------------ | ------------------------------------------------------------ |
| إطار التطبيق       | TanStack Start، TanStack Router، React 19، Vite              |
| الواجهة            | Tailwind CSS، Radix UI، Framer Motion                        |
| البيانات والمصادقة | Firebase Authentication، Cloud Firestore، Cloud Storage      |
| واجهات الخادم      | مسارات TanStack Start تحت `src/routes/api/` (على **Vercel**) |
| المهام الخلفية     | Firebase Cloud Functions (بريد وجداول وصيانة)                |
| الذكاء الاصطناعي   | Google Gemini (`@google/genai`) على خادم Vercel              |
| الفيديو            | Daily.co                                                     |
| الدفع بالبطاقة     | Polar                                                        |

## بدء سريع (للمطورين)

```bash
npm install
# أنشئ ملف .env.local وفق أسماء المتغيرات في docs/DEPLOYMENT.md
npm run dev
```

يفتح المتصفح عادة على `http://localhost:5173` أو العنوان الذي يظهر في الطرفية.

---

## هيكل المستودع (مختصر)

```
src/routes/          # الصفحات والتخطيطات (توجيه ملفّي)
src/routes/api/      # معالجات HTTP للخادم فقط
src/lib/             # مساعدو Firebase والأنواع وأدوات الذكاء الاصطناعي
functions/           # Cloud Functions (محفزات وجداول)
firestore.rules      # قواعد أمان Firestore
storage.rules        # قواعد التخزين
docs/                # البنية والميزات والنشر
```
